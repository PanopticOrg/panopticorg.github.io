# Créer ses propres plugins

Si les plugins déjà existant ne correspondent pas à vos usages, n'hésitez pas à créer les votres, ils consistent en grande partie en du code python libre dans lesquels vous écrirez le comportement souhaité. Il suffit de respecter un format et de connaitre les quelques fonctions panoptic permettant de se raccorder aux actions de l'interface. 

Un guide détaillé avec exemple est disponible ici: 

https://github.com/CERES-Sorbonne/Panoptic/wiki/Plugin
